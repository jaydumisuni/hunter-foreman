# Hunter Foreman Architecture

Hunter Foreman is presented publicly as an AI operations foreman for small businesses.

## Public Demo Components

1. AI receptionist receives a customer request.
2. Intent router classifies the request.
3. Orchestrator creates a task or workflow.
4. Dashboard shows status and confidence.
5. Handoff layer prepares email or WhatsApp actions.
6. Human escalation catches low-confidence or sensitive requests.

## Private Boundary

The public hackathon version excludes private THETECHGUY internals, repair engines, client data, credentials, and proprietary workflows.

## Submission Principle

The judge-facing project must be understandable, containerized, and safe to run from a public GitHub repository.
